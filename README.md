## 🚀 AI-Powered Invoice Processing Pipeline (n8n + LlamaIndex + Groq)

An end-to-end automated invoice processing system built with **n8n workflows**, **LlamaIndex**, and **Groq AI**, designed to eliminate manual data entry and streamline financial document processing.

### ⚡ What it does

* 📁 Automatically triggers when a PDF invoice is uploaded to Google Drive
* 🧠 Uses **LlamaIndex** to parse and structure invoice content
* 🤖 Leverages **Groq LLM (120B)** to extract key fields:

  * Invoice Number
  * Invoice Date
  * Vendor Name
  * Tax Amount
  * Total Amount
* 📊 Appends structured data directly into Google Sheets
* 📂 Moves processed files to archive folder and cleans up originals

### 🔥 Key Features

* Fully automated end-to-end pipeline (zero manual steps)
* Sub-30 second invoice processing time
* Scalable workflow using n8n visual automation
* AI-powered document understanding
* Easy integration with Google Workspace

### 📈 Impact

* Reduced invoice processing time from **15 minutes → < 30 seconds**
* Saved **4+ hours in the first week**
* Eliminated manual data entry errors
